# Urban Shadow Analysis, Magdeburg

Satellite image segmentation and shadow modelling in Magdeburg, Germany.

---

## Phases

1. Satellite imagery acquisition
2. Urban segmentation (buildings + vegetation) 
3. Shadow modelling

---

## Project structure

```
urban-shadow-analysis/
├── notebooks/          # exploratory notebooks
├── src/                # reusable modules (segmentation, shadow, utils)
├── data/               # raw + processed data
├── references/         # papers and external resources
├── README.md
└── requirements.txt
```
